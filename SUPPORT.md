# 🆘 SUPPORT
Support & aide — Samsung Devices & OS Knowledge Suite

Merci de votre intérêt pour ce projet. Voici comment obtenir de l’aide.

---

## 📚 Documentation
Avant toute demande, consultez la documentation :
- `/docs/overview`
- `/docs/devices`
- `/docs/os`
- `/docs/security`

---

## 🐞 Signaler un bug
Utilisez le template dédié :
- `.github/ISSUE_TEMPLATE/bug_report.md`

Merci d’inclure :
- étapes pour reproduire
- environnement
- fichiers concernés
- logs éventuels

---

## 🌟 Demander une fonctionnalité
Utilisez :
- `.github/ISSUE_TEMPLATE/feature_request.md`

Expliquez clairement :
- le besoin
- la valeur ajoutée
- les alternatives possibles

---

## 💬 Questions générales
Ouvrez une issue avec le template :
- `.github/ISSUE_TEMPLATE/issue_template.md`

---

## 🤝 Contribution
Consultez :
- `CONTRIBUTING.md`
- `GOVERNANCE.md`

---

## 🔐 Sécurité
Pour signaler une vulnérabilité :
- ne créez **pas** d’issue publique
- suivez les instructions dans `SECURITY.md`

---

## 📬 Contact
Pour toute autre demande, utilisez les discussions GitHub ou les issues.
