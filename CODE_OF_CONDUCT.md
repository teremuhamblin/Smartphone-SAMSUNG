📘 CODE_OF_CONDUCT.md

Code de conduite — Samsung Devices & OS Knowledge Suite

---

🧭 1. Notre engagement

Nous nous engageons à faire de ce projet un espace :

- accueillant  
- respectueux  
- inclusif  
- collaboratif  
- professionnel  

Chaque membre de la communauté, qu’il soit contributeur occasionnel ou mainteneur actif, doit pouvoir participer sans crainte d’être jugé, discriminé ou harcelé.

---

🤝 2. Comportements attendus

Pour maintenir un environnement sain, nous attendons de chacun :

✔ Respect
Traiter les autres avec considération, même en cas de désaccord.

✔ Communication constructive
Exprimer ses idées clairement, écouter les autres, éviter les attaques personnelles.

✔ Collaboration
Aider les nouveaux contributeurs, partager les connaissances, favoriser l’amélioration continue.

✔ Professionnalisme
Rester factuel, éviter les propos agressifs, respecter les décisions collectives.

✔ Inclusion
Encourager la participation de tous, indépendamment de l’expérience, du niveau technique ou de l’origine.

---

❌ 3. Comportements inacceptables

Les comportements suivants ne seront pas tolérés :

- Harcèlement, intimidation, moqueries  
- Propos discriminatoires (origine, genre, orientation, religion, handicap…)  
- Attaques personnelles ou insultes  
- Diffusion de contenu offensant ou inapproprié  
- Spam, publicité non sollicitée  
- Sabotage du projet ou des discussions  
- Publication d’informations privées sans consentement  

---

🛡️ 4. Responsabilités des mainteneurs

Les mainteneurs du projet ont pour rôle :

- d’appliquer ce code de conduite  
- de modérer les discussions  
- de prendre des mesures en cas de comportement inapproprié  
- d’expliquer clairement les décisions prises  
- de protéger la communauté  

Ils peuvent :

- supprimer des commentaires  
- fermer des issues  
- refuser des contributions  
- bannir temporairement ou définitivement un utilisateur en cas de violation grave  

---

🧩 5. Signalement d’un problème

Si vous êtes témoin ou victime d’un comportement inapproprié :

- ouvrez une issue privée si possible  
- ou contactez directement un mainteneur du projet  

Les signalements seront traités :

- rapidement  
- confidentiellement  
- avec impartialité  

Aucune forme de représailles ne sera tolérée.

---

📜 6. Application du code

Toute violation du code de conduite peut entraîner :

- un avertissement  
- une suspension temporaire  
- une exclusion définitive du projet  

Les décisions seront prises en fonction de la gravité et du contexte.

---

🌐 7. Portée

Ce code de conduite s’applique :

- au dépôt GitHub  
- aux issues  
- aux pull requests  
- aux discussions  
- aux échanges privés liés au projet  
- aux espaces communautaires associés  

---

🤝 8. Engagement final

En participant à ce projet, vous acceptez de respecter ce code de conduite et de contribuer à un environnement collaboratif, professionnel et bienveillant.
